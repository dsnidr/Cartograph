# Modded World Performance

**System specs**:
- 32GB Memory
- M.2 NVME Drive
- Ryzen 9 9900X (12 cores, underclocked)

**World size**: 3584x3584 centered on 0,0

Run from idle.

| Scale            | Time (ms)  | Avg. CPS  | File Size  |
|------------------|------------|-----------|------------|
| full `1:1`       | 7,513      | 57,540    | 46.9 MB    |
| half `2:1`       | 4,900      | 58,688    | 15.5 MB    |
| thumbnail `16:1` | 4,147      | 59,426    | 4.48 MB    |

**CPS**: chunks per second.

The full and half image variants were excluded from the repo since they are quite large.
